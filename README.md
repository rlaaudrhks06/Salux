# Salux

1. 프로젝트 개요

웨어러블 디바이스 기술과 인공지능 기술의 발전으로 개인 맞춤형 헬스케어 서비스에 대한 수요가 증가하고 있다. 기존 스마트워치는 단순한 생체 데이터 수집 및 통계 분석에 머무르는 한계가 있으며, 개인의 생체리듬과 생활 패턴을 정밀하게 반영하지 못한다.
본 프로젝트는 인공지능(AI)을 활용하여 사용자의 개인 생체리듬을 학습·분석하는 스마트 바이오 워치를 개발하는 것을 목표로 한다. 심박수, 수면 패턴, 활동량, 체온, 혈중산소, 혈압 등의 생체 데이터를 지속적으로 수집하고, AI 알고리즘을 통해 개인 맞춤형 생체리듬 모델을 구축한다. 또한 개인의 생체리듬의 이상이 있는 경우 원인과 이 이상을 해결할 수 있는 조언을 한다. 
ex) 산책 30분 정도 해야될 것 같아요, 물을 마시는게 좋겠어요, 휴식이 필요해요.
이를 통해 사용자의 건강 상태를 예측하고, 일상생활의 효율성과 장기적인 건강 관리를 지원한다.

2. 개발 목표
AI를 활용한 실시간 생체 데이터 분석
머신러닝 기반 개인 맞춤 생체리듬 모델 구축
AI 기반 건강 알림 및 맞춤형 추천 제공

3. 주요 기능
- 개인 맞춤형 생체리듬 학습
     사용자 고유의 생활 패턴과 신체 변화를 AI가 지속적으로 학습

- 지능형 건강 모니터링
     심박수, 혈중산소, 혈압, 휴식 등을 실시간 분석

- 스마트 알림 제공
     운동, 휴식, 수분 섭취, 수면 시간 추천
